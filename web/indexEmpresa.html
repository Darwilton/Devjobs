¿Que es Devjobs?

Devjobs es una aplicación web que le proporciona una bolsa de empleo especializada en el área de desarrollo de software a el usuario y a su vez le permite a las empresas publicar las ofertas de empleo. Por otra parte, los desarrolladores podrán buscar las ofertas publicadas y aplicar su hoja de vida a dichas ofertas. La aplicaciòn mostrara una breve descripción de las ofertas, el perfil requerido, el salario ofrecido y la empresa que publica la oferta. Las hojas de vida aplicadas a una oferta seràn enviadas en una cola de prioridad a las empresas ofertantes

Desarrollador

Eres el motor de la empresa aquel que aporta las ideas, la creatividad y el tiempo para llevar a cabo una o varias tareas, tu puedes desarrollar varios trabajos al mismo tiempo e incluso completarlos en el tiempo requerido para poder entregarlos, si estas aquí es porque de alguna manera tu anterior empresa no supo apreciar tus cualidades o simplemente eres nuevo, sea cual sea dale al botón “Registrarse Como Desarrollador” para empezar.

Empresa

Eres una maquinaria compleja que necesita de empleados para que todo funcione correctamente, pero sea cual sea tu servicio o producto también necesitas un desarrollador, ya sea para arreglar una de tus hojas de cálculo de excel o para crear una página web para que tengas más compradores y expandir tu público, entonces si necesitas un desarrollador dale click al botón “Registrarme Como Empresa” para empezar.
